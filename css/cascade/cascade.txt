 What is the Cascading Order in css?
  or
 What style will be used when there is more than one style specified for an 
 HTML element?
  All the styles in a page will "cascade" into a new "virtual" style sheet
  by the following rules,
  where number one has the highest priority:
    1.Inline style
	2.Internal style
	3.External style sheets
	4.Browser default style
 
 so, an inline style has the highest priority, and will override external 
 and internal styles and browser defaults.

FAQs:
 Which style is priority among the three styles?
 Inline style is the first priority.

 Which style is recommended to use in html code?
  External style sheet is the recommended to use.
