          HTML ELEMENTS
  What are the html elements?
   HTML elements are basic build blocks of html page.
   Element is defined by start tag, some content and end tag.
   Element is everything from start tag to end tag.
   Syntax:
     <tagname>content goes here..</tagname>
   Example:
    <h1>This is a heading</h1>
	<p>this is a para.</p>
=====
 What are Nested elements?
  HTML elements can be nested, means that elements can contain other
  elements.
  All html documents consists of nested HTML elements.
  Example:
    <!DOCTYPE html>
	 <html>
	    <head>
		    <title>
		      Elements
	    	</title>
		  </head>
		<body>
		    <h1>
		      This is a heading
		    </h1>
			  <p>
			  This is a para
		    </p>
		</body>
		     <footer>
		     </footer>
	 </html>
=====
Example Explained:
   1.The <html> elements is the root element and it defines the 
     whole html document.
   2.It has a start tag <html> and an end tag </html>.
     Then, inside an <html> element there is an <body> element.
   3.It has a start tag <body> and an end tag </body>.
     Then inside a <body> element there are two other elements:
     <h1> and <p>.
   4.The <h1> element defines a heading.
     It has a start tag <h1> and an end tag </h1>.
   5.The <p> element defines a paragraph.
     It has a start tag <p> and an end tag </p>.
   6.The <footer> element defines the document's footer.
     It has a start tag <footer> and an end tag </footer>.
====
 Never skip the End tag.
   Some HTML elements will display correctly, even if you forget the 
   end tag.
   However, never miss end tag, if you miss, an unexpected results and 
   errors may occur.
 ===
 What are empty HTML elements?
  HTML elements with no content are called empty elements.
  The <br> tag defines a line break, with an empty element without
  a closing tag.
  Example:
   <!DOCTYPE html>
    <html> 
        <head>
           <title>Empty HTML elements</title>
        </head>
        <body>
           <p> this is a <br> paragraph with line break.</p>
           <br>
        </body>
    </html>
======
HTML is not case sensitive
  HTML tag are not case sensitive:
    <p> means the same as <p>.
	but always use lower case tag names.
