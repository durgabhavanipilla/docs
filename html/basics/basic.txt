       
How to view HTML source code?
  Right click in an HTML page.
  select "view page source"(in chrome).
  select "view source"(in Edge).
  or similar in other browsers.
  This will open a window containing the HTML source code of the page.
=====
 How to inspect an HTML element?
  Right click on an element (or a blank area).
  Choose "Inspect" or "Inspect Element" to see what elements are made up of, 
  you can also edit the HTML or CSS on-the-fly in the elements 
  or styles panel that opens. 
  Whatever the changes tnat you did in inspect mode those will not
  reflecte in source code, changes are temperary only.
=====
 What will happen when you open html page from browser?
  Your browser will read entire html page at a time, it will interpret the
  html document, and converted into user understandable formate.
===== 
  If any error in html code in interpretation phase, your browser won't stop
  the interpretation at the particular line, it will print error in console
  then it will interpret next line.
